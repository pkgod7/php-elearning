Every full C program begins inside a function called main. A function is simply a collection of commands that do something. The main function is always called when the program first executes. From main, we can call other functions, whether they be written by us or by others or use built-in language features. To access the standard functions that comes with your compiler, you need to include a header with the #include directive.
